# SkillFront Free Certification Programs

Official GitHub repository for the SkillFront GitHub Pages site that lists selected free certification programs, registration links, and companion audiobook study resources.

## Live Site

The intended live site URL is:

`https://skillfront.github.io/`

## Purpose

This repository is designed as a clean public navigation hub for selected SkillFront free certification programs. It helps visitors:

- compare programs quickly
- open the official registration pages
- access companion YouTube audiobook study resources

## Repository Structure

- `index.html` → primary live homepage
- `index.md` → alternate lightweight markdown homepage source
- `README.md` → repository overview and maintenance notes

## Deployment Notes

Use `index.html` as the live homepage of the GitHub Pages site.

Keep `index.md` as an alternate markdown version. Do not use both as the active root homepage at the same time.

## Maintenance

When updating the site:

1. edit `index.html`
2. keep summaries factual and useful
3. link directly to official SkillFront pages
4. avoid keyword stuffing or thin promotional copy
5. commit changes to the `main` branch

## Official Positioning

This repository is intended to function as an official SkillFront public resource page for selected free certification programs and study resources.
